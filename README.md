## Системные требования
- Операционная система: Windows 7/8/10/11
- Разрядность: x64 / x86
- Дополнительно: Не требует установки дополнительных библиотек

## Использование
1. Запустите `DosToolVB.exe`.
2. Выберите сайт для тестирования (например, Google, YouTube).
3. Нажмите соответствующую цифру в меню.
4. Для остановки теста нажмите `Ctrl + C` в окне консоли/закройте программу.
5. Для возврата в меню заново запустите приложение.
Внимание!: Используйте только на своих серверах или с разрешения владельца!
